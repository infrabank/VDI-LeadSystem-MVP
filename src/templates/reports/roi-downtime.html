계산 공식

연간 총 다운타임 = 평균 중단시간 x 연간 장애 횟수

연간 생산성 손실 = 사용자 수 x 시간당 인건비 x 연간 다운타임

개선 후 손실 = 연간 손실 x (1 - 복구시간 단축률)

연간 절감액 = 현재 손실 - 개선 후 손실

예상 투자비 = 사용자 수 x 300,000원 (사용자당 가정)

회수기간 = 예상 투자비 / 연간 절감액
